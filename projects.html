Comic Get It

Built with AngularJS, this app lets a user log in to their profile using Firebase Authentication to create a digital catalogue of their comic-book library. The user is able to search for comicbooks or characters they are interested in, which is then ran through comic-vine's API. With the results, the user can rate and add items to their current comic list or begin a wishlist for titles not yet owned. The functionality for creating a list of favorite heroes and villains is a feature as well, complete with a rating system.

Chore Score

An app designed for parents trying to incentivise the mundane chore, for roommates looking to compete against each other, or for motivating yourself around the house, ChoreScore aims to make you want to complete a daily task. Each chore is created with a point value, which is given to the user upon completion. This can then be redeemed for rewards that range in point value.
